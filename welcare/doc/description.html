Main Features

Navigation Menu

There are two different menu-styles

Mega Menu

Mega-menu is designed according to 960 grid. It is a full length menu having different contents in it.

Custom Menu

Another style of navigation menu is set according to custom width.

Main Slider

The main slider of the Welcare is designed in jquery and CSS3 properties using transparency, transformation, opacity, transition etc. The slider is well structures as the user don't have to face any difficulty to add more slides in it.

Symptom Checker

Welcare is designed for medical industry, it has the feature of Symptom Checker. The user can check the symptoms of disease. There are three main steps to diagnose the disease. These three steps are following:

  • There is a button 'Start here' on the slider. To proceed, user has to click on the button. The user can view the front and back view of the gender he selected.

  • Next step is to select the gender i.e. male, female or kid. There is an agreement to accept by the user. Click next and the user will be guided to the next step.

  • To proceed further, user first selects the part of body for which he wants to check the symptoms. After selecting the part of body, user is directed to choose the part of body of which he wants to check symptoms. This leads the user to the select the symptoms.

Color Picker

There is a color picker which allows the user change the color of containers. User can change the background color of page, Buttons, headings, links etc.

Various Width layouts

Welcare is a template that has enhanced functionality, offering the user to change the width layout.

Tab area has 4 different width styles that change the content according to the given width. Different width styles can be applied by giving the 'col-1', 'col-2', 'col-3', 'col-4'.

Blog areas have 3 different width styles. In each width styles, contents are well-organized. Each content in blocks is completely independent as user just has to change the parent class and the whole structure comes in a different style.

Responsive Layout

Welcare is designed considering various devices such as iPads, notebooks etc making it compaltible for all devices.

Gallery View

he template provides user different gallery views; list view and grid view. Another style of gallery page is gallery with a sidebar,having necessary contents there.

User can view gallery in two grid and list style.


Grid View

Grid view includes the name, author name and the post title.

List view

List view gives a complete list of images. There is a small section on the right side of the gallery that includes a brief description, respective category and rating of that image.

Gallery with Sidebar

Cross-Browser Compatibility

Aside from being a joy to use, Welcare is a very well-structured template, compatible to all the browsers.

Plug-in & Jquery UI

Welcare has been designed using some impressive widgets and animation effects including scroll and many more.

Clean Markup

Script behavior is separated from page presentation. This separation leads to the cleaner and maintainable code, providing ease to user to customize easily.
