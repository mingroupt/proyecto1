How to change the width Layout of Navigation Menu

User can change the navigation menu style in two different styles. One is mega-menu, set according to the 960 grid. The other one is designed in various widths as user can change the width of sub-menu by just giving the column-number. User can change the menu style according to 960 grid by giving the class 'dropdown_full_column' to a div in the li.

Size can be decreased by giving a class 'dropdown_column' into a div in the li 'column'. In the div of the li, column-number is given so that the user can change the width of block according to him. User can also change the size of the columns by giving different sizes from 'column-1' to 'column-8', lesser the column-number, smaller is the width.

Change Content Layout of Sub-menu

User can change the layout of sub-menus. This is easy as the user just has to change the name of respective div. Change the name from 'col_2' to 'col_5' and a different layout of sub-menu is appied. Similarly, user can apply these classes from 'col_1' to 'col_8', greater the ''col_(number)', width will b increasing.
